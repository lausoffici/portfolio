## Dependencies and tools
- Next.js
- Tailwind CSS
- ESLint

## Usage

### Production
```
> npm run build
```
```
> npm start
```

### Development
Install project dependencies
```
> npm install
```

Start development server
```
> npm run dev
```

Show errors/warnings detected by ESLint
```
> npm run lint
```
